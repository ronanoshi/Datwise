## HOW TO RUN

### QUICK START (Windows)
  1. Double-click: setup.bat
  2. Terminal 1: cd Datwise.Api && dotnet run
  3. Terminal 2: cd Datwise.WebForms && dotnet run
  4. Open: https://localhost:7290

### QUICK START (Mac/Linux)
  1. Terminal: chmod +x setup.sh && ./setup.sh
  2. Terminal 1: cd Datwise.Api && dotnet run
  3. Terminal 2: cd Datwise.WebForms && dotnet run
  4. Open: https://localhost:7290

### MANUAL SETUP
  dotnet restore
  dotnet build

  Then run each project in separate terminals:
  - Datwise.Api: dotnet run (uses port 53486)
  - Datwise.WebForms: dotnet run (uses port 7290)
